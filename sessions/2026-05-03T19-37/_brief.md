# 📋 작업 브리프

**원 명령:** 🔥 BUSINESS CONTEXT (AI 입력용)
1. 사업 개요
브랜드명: 온잇 (ONIT)
사업 유형: 이커머스 기반 상품 소싱 및 판매 + 퍼포먼스 마케팅
판매 채널: 쿠팡 (로켓그로스 중심)
광고 채널: 메타 광고 (Facebook / Instagram)
수익 구조:
상품 마진 (주 수익)
쿠팡파트너스 수수료 (보조 수익, 광고비 방어용)
2. 사업 모델 구조

1688 저원가 상품 소싱 → 쿠팡 등록 → 메타 광고로 트래픽 유입 → 쿠팡파트너스 링크 활용 → 상품 판매 + 수수료 수익

핵심 전략:

저원가 상품을 고마진 구조로 전환
광고 + 전환 최적화로 수익 구조 구축
테스트 → 데이터 → 집중 → 확장
3. 현재 상태
초기 단계 (테스트 진행 중)
자본: 제한적
목표:
1~2개월 내 매출 구조 구축
단기: 월 매출 100만원
중기: 월 매출 1,000만원
장기: 월 매출 5,000만원
4. 상품 전략

현재 카테고리:

액세서리 (목걸이, 팔찌)
선글라스
가방
쿨토시 등 테스트 상품

전략:

저가로 테스트 → 반응 나오면 고가 구조로 전환
단품이 아니라 세트/패키지로 고가화
“보여지는 가치” 중심 판매

상품 선택 기준:

눈에 보이는 변화 (착용 전/후 가능)
충동구매 가능
사진/영상으로 설득 가능
클레임 낮은 상품
5. 마진 구조
목표 마진율: 50% 이상

순마진: 5,000~20,000원

## 요약
현재 진행 중인 상품 테스트의 전환율 개선을 위해 데이터 기반으로 최적화된 광고 소재 및 상세페이지 초안 생성을 요청합니다.

## 분배
- **💰 Business**: 현재 테스트 중인 상품 목록(액세서리, 선글라스 등) 중 가장 잠재력이 높거나 데이터가 부족한 3가지 상품을 선정하고, 해당 상품에 대한 고마진 전환 구조를 재설정할 것.
- **🔍 Researcher**: 선정된 3가지 상품 카테고리 내에서 경쟁사 상세페이지 및 메타 광고 소재의 성공/실패 요인을 리서치하고, 현재 온잇(ONIT)이 부족한 부분을 분석하여 핵심 인사이트를 도출할 것.
- **✍️ Writer**: 리서치 결과를 바탕으로 가장 잠재력이 높은 상품 1가지에 대해 '고가 세트/패키지' 판매 전략을 적용한 고마진 상세페이지 카피와 메타 광고용 후크(Hook) 문구 5안을 작성할 것.
- **🎨 Designer**: Writer가 제시한 카피 및 핵심 가치에 기반하여, 선정된 상품에 적용할 고마진 제품의 비주얼 콘셉트(컬러 팔레트, 레이아웃 방향)를 정의하는 디자인 브리프 초안을 작성할 것.
